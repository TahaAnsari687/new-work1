What Does Ms 3dxara Mean?

Xara 3D Maker is a utility for creating still or animated 3D graphics and text, such as headings, titles, logos and buttons. It is ideal for on-screen graphics such as web pages, mailshots, movie titles and credits and presentations, and you can even save any animation as a 3D screensaver. Put simply, Xara 3D Maker applies a 3D extrusion effect to a given piece of text or shape. You can then customize the graphic - for example change the lighting, bevel, color, animation style, texture and font of the text, as well as the 3D extrusion used. The output is fully anti-aliased for the best possible on-screen quality. All this is controlled with a simple and intuitive user interface that's very easy to get the hang of
